## Getting Started

### Installation

```bash
sh -c $(curl -L https://raw.githubusercontent.com/UnityTheCoder/Voxel/main/assets/install.sh)
```
OR
```
make install
```



## Usage
```bash
python3 src/interpreter.py -h
```
OR
```bash
vxc -h
```

## Update
```
vxc --update
```
OR
```
python3 src/interpreter.py --update
```
